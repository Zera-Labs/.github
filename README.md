# .github

Organization-wide community health files and profile for [ZeraLabs](https://github.com/zeralabs).

## Contents

```
profile/README.md        → Organization profile (shown on github.com/zeralabs)
CONTRIBUTING.md          → Default contributing guidelines
CODE_OF_CONDUCT.md       → Community code of conduct
SECURITY.md              → Vulnerability reporting policy
FUNDING.yml              → Sponsorship configuration
ISSUE_TEMPLATE/          → Default issue templates (bug report, feature request)
PULL_REQUEST_TEMPLATE.md → Default PR template
```

These files apply as defaults across all repositories in the org unless overridden by a repo-specific version.
